A **team manager** (also called the team captain) is the player responsible for creating and operating a team on C3 Esports. Every team has exactly one manager at a time. Managers have elevated permissions that regular players do not: they can register teams for seasons, manage the roster, submit match results, and file disputes.

<Note>
  Being a manager is a commitment. If you go inactive during a season, your team may forfeit matches. Make sure you are prepared to be available on match days.
</Note>

## How to become a manager

Any registered C3 Esports user can create a team and become its manager — just navigate to [c3esports.com/teams/create](https://c3esports.com/teams/create).

<Tip>
  Before creating a team, make sure you have completed your own account setup: verified your .edu email and linked your Discord account. You need both to compete, and your players will need them too.
</Tip>

You can also become a manager if an existing manager **transfers** the role to you. See [Team Settings](/managers/team-settings) for details.

## Manager vs. player permissions

| Action | Manager | Player |
|---|---|---|
| Submit match results | Yes | No |
| Add/remove players from roster | Yes | No |
| Register team for a season | Yes | No |
| File a dispute | Yes | No |
| Check in to a match | Yes | Yes |
| View match schedule | Yes | Yes |
| View standings | Yes | Yes |
| Edit team settings | Yes | No |
| Transfer manager role | Yes | No |
